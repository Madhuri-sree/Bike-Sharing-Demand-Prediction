**Bike Sharing Demand Prediction**

Overview
This project predicts the total number of bike rentals using a Random Forest Regression model. It uses historical bike-sharing data along with weather and seasonal information to estimate bike rental demand.

Features
- Data preprocessing and cleaning
- Feature selection
- Train-test split
- Random Forest Regression
- Model evaluation using Mean Squared Error (MSE) and R² Score
- Visualization of actual vs. predicted bike rentals

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Dataset
**Bike Sharing Dataset**

Source: https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset

Files used:
- day.csv
- hour.csv

How to Run
1. Download or clone this repository.
2. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```
3. Open `Bike_Sharing_Demand_Prediction.ipynb`.
4. Make sure `day.csv` is in the same folder as the notebook.
5. Run all cells.

Machine Learning Model
- Random Forest Regressor

Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score
